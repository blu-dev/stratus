# stratus - Mod Loader for Super Smash Bros. Ultimate

Stratus is the spiritual successor to [ARCropolis](https://github.com/Raytwo/ARCropolis).

ARCropolis, while driving the SSBU Modding scene for years now, suffers from a legacy codebase that is difficult to maintain.

## Improvements over ARCropolis
- Uses less memory overhead for hash lookups for logging using [smash-hash](https://github.com/blu-dev/smash-hash)
- More to be written later (ghidra was analyzing something while I was writing this)
